# To-Do List Manager

A simple and user-friendly command-line To-Do List Manager developed in Python. This project helps users organize their daily tasks efficiently by allowing them to add, view, complete, and delete tasks. All task data is stored permanently in a JSON file, ensuring that tasks remain available even after the program is closed.

## Features

* Add new tasks with unique IDs
* View all saved tasks
* Mark tasks as completed
* Delete tasks from the list
* Automatic data saving using JSON
* Persistent storage (tasks remain saved after restarting the program)
* Input validation for better user experience

## Technologies Used

* Python
* JSON File Handling
* File Management (OS Module)

## How It Works

1. The program loads existing tasks from a JSON file when it starts.
2. Users can add new tasks through the menu.
3. Tasks can be viewed along with their completion status.
4. Users can mark tasks as completed.
5. Unwanted tasks can be deleted.
6. All changes are automatically saved to the JSON file.
